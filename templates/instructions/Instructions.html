Instructions

In this study, you will be shown a series of pictures like the one below. Your task is to guess the number of BLUE circles minus the number of YELLOW circles. You will enter your answer using a slider. If you thought there were 2 more BLUE circles than YELLOW circles, you would answer 2. Whereas, if you thought there were 2 fewer BLUE circles than YELLOW circles, you would answer -2.

You will only be given 2 seconds to view each picture, so make sure you are ready before beginning each question.

For example, here there are 17 BLUE circles and 33 YELLOW circles, so the correct answer is -16. You do not need to be precise to get the question correct. Your answer will be counted as correct if it is within +3 or -3 of the correct answer. Here, answering -13, -14, -15, -16, -17, -18, or -19 would be marked as a correct answer.

You will complete the study in a pair with another participant. You will either be player 1 or player 2.

On each question, player 1 and player 2 will see the same image, so the correct answer will be the same for both of them.

You will see example questions on the next page.
